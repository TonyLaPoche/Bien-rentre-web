# Contact - Formulaire de support

## Titre de section
Contactez-nous

## Sous-titre
Vous avez des questions ? N'hésitez pas à nous contacter.

## Formulaire de contact
### Champs
- **Titre** : Champ texte requis
- **Message** : Zone de texte multiligne requis
- **Adresse email** : Champ email requis

### Bouton d'envoi
Envoyer le message

### États du formulaire
- **Par défaut** : Bouton "Envoyer"
- **Envoi en cours** : Bouton "Envoi en cours..." (désactivé)
- **Succès** : Message "Message envoyé avec succès !"
- **Erreur** : Message "Erreur lors de l'envoi. Veuillez réessayer."

## Service email
- **API utilisée** : EmailJS
- **Configuration** : Service ID, Template ID, Public Key
- **Destinataire** : contact@antoineterrade.com

## Style
- Formulaire centré avec ombre
- Champs avec bordures arrondies
- Validation en temps réel
- Messages d'état colorés (vert/rouge)
- Animation de chargement sur le bouton
