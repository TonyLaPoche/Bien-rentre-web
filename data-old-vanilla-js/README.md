# Bien-Rentré - Application de sécurité nocturne

Site vitrine pour l'application mobile Bien-Rentré, qui permet de partager sa position GPS en temps réel avec des contacts de confiance lors de déplacements nocturnes.

## Structure du site

- **Hero Section** : Présentation principale de l'application
- **Fonctionnalités** : Les 4 étapes principales d'utilisation
- **Comment ça marche** : Explication du processus
- **FAQ** : Questions fréquemment posées
- **Contact** : Formulaire de support
- **Navigation** : Menu responsive
- **Footer** : Liens légaux et copyright

## Technologies utilisées

- **Application mobile** : Flutter/Dart
- **Backend** : Firebase (authentification, base de données)
- **Notifications** : Push notifications en temps réel
- **GPS** : Géolocalisation précise
- **Sécurité** : Chiffrement des données

## Fonctionnalités clés

- Partage de position GPS en temps réel
- Gestion de contacts de confiance
- Notifications automatiques
- Interface intuitive
- Respect du RGPD

## Pages légales

- Conditions d'utilisation
- Politique de confidentialité

## Déploiement

Le site est optimisé pour GitHub Pages avec un build statique.
